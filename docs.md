# 教师绩效考核系统 - 用户操作手册与技术文档

## 1. 系统概述
本系统是专为“声度星主持”机构设计的教师绩效考核工具，旨在通过标准化的考核维度和自动化的计算逻辑，公正、高效地评估教师的工作表现。

## 2. 考核维度说明
系统包含以下六个核心考核维度：
- **续费率 (35%)**: 评估教师对存量学员的维护能力。
- **出勤率 (30%)**: 考核教学交付的稳定性。
- **体验课转化率 (15%)**: 评估新学员的转化能力。
- **回访数 (10%)**: 考核家校沟通的频次与质量。
- **宣传数量 (10%)**: 评估教师对机构品牌的推广贡献（朋友圈、小红书）。
- **加分项 (上限15分)**: 包含转介绍、参赛获奖和视频拍摄等额外贡献。

## 3. 操作指南
1. **进入系统**: 在数据仪表盘首页点击顶部导航栏的“绩效考核”按钮。
2. **输入教师信息**: 在左侧侧边栏下方输入被评估教师的姓名。
3. **分维度评估**: 
   - 从左侧菜单依次选择考核维度。
   - 阅读页面顶部的“考核规则说明”。
   - 在“数据录入”区域输入各项实际数据。
   - 观察“实时计算过程”以确认得分逻辑。
4. **查看汇总**: 完成所有维度评估后，点击左侧下方的“考核汇总报告”。
5. **保存与导出**:
   - 点击“保存评估数据”将结果存入浏览器本地存储。
   - 点击“导出PDF报告”可进行打印或另存为文件。

## 4. 技术架构
- **前端框架**: Vue 3 (Composition API)
- **样式方案**: Tailwind CSS
- **数据处理**: Day.js (日期处理)
- **数据持久化**: localStorage (本地存储)
- **计算逻辑**: 核心算法封装在 `performance.js` 中的 `validateAndCalculate` 函数。

## 5. 精度与验证
- 所有分数计算均保留 1-2 位小数。
- 系统对负数输入进行了实时拦截验证。
- 逻辑严格遵循提供的 12月绩效明细规则图片。
